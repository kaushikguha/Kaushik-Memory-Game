Information about my Memory Game

This is a simple memory game

The user can Click on the cards to turn them over

Each card has a letter from the English alphabet (A to Z)

Once the user matches two cards, the cards remain revealed

The user starts at level 1 and progresses to level 3

The difficulty of each level will also be variable

If the user cannot match all the available cards in the given time the player loses the game

Two aspects of the users performance is kept track of

1. The no. of clicks 2. The amount of time spent
